Entramos al mundo GIT

Que es git ?

Git es basicamente un sistema de control de versiones distribuidas que estas diseñadas para dar ayuda a los proyectos que se crean con una gran velocidad de respuesta y facilidad y una gran gama de configuraciones para luego sincronizar con el git hub Git es libre y de codigo abierto por lo tanto puedes hacer muchas cosas sin ningun problema ni restricciones Su facil empleo y uso hace este sistema como uno de los mas usados y mas recomendados!

Con git no se necesita un repositorio central, cada miembro de nuestro equipo tiene una copia identica de los proyectos que hemos hecho con todo su contenido ya que es muy util porque podemos trabajar con nuestro trabajo y guardarlo en nuestro repositorio local para no tener problemas en el futuro y hacer todos los cambios que queramos y en cualquier momento podemos hacer una copia de nuestro trabajo y enviarsela a nuestros demas compañeros o viceversa y luego subirlo finalmente al git

Instalacion

Vamos a la consola y ponemos.

$ sudo apt-get install git

Ahora para saber si se ha instalado y la version que tiene

$ git --version

Configuracion

Para introducir datos manualmente como el nombre y el correo electronico

$ git config --global user.name

Para poner el correo electronico

$ git config --global user.email

Puedes comprobar como ha quedado los datos que hemos introducido

$ git config --global --list

AYUDA

Si necesitas ayuda o quieres saber cuales son los comandos y que hacen cada uno tenemos el comando

$ git help

Creando Repositorio y el primer commit

Para comenzar con git lo que tenemos que crear es una carpeta .git qye es donde tendremos todos los datos etc de git

$ Git init

Despues de hacer un git init podemos hacer un commit para ver el estado del repositorio con los commit

Para hacer un commit hacemos

$ git add . y git commit -m "AQUI VA EL TITULO QUE SE VERA EN EL COMMIT"

Para ver el estado

$ git status

Comits

Si queremos saber cuantos commits hemos hecho o si habido un error

Ponemos en la consola

$ git log

Si queremos mas informacion

$ git log --oneline --decorate --graph

Clonar

Una vez que ya sabemos crear un repositorio nuestro y hacer commit. Vamos a clonar un repositorio de algun colaborador nuestro porque hemos empezado un nuevo proyecto y lo ha creado el para clonar

Vamos a la consola y ejecutamos

$ git clone y la url de donde queremos clonar el trabajo

Nota: Podemos tener dos directorios el directorio principal que es el git(.git) y luego el directorio de trabajo(Working Directory)

Elimiar contenido

En caso de haber hecho un fallo o un cambio que no querias hacer puedes eliminar los datos del commit

Vamos a la consola y ejecutamos

$ git reset --hard

CREAR RAMA

Vamos a crear una rama nueva esto esta bien porque si ya hemos hecho commits estarian apuntando en la rama master pero si quieremos practicar y no tener problemas podemos hacer una rama para practicar y asi no tener problemas por si nos equivocamos en un futuro

Usamos el comando git branch y el nombre que le queremos poner

$ git branch ramaPrueba

Para cambiarnos a la rama que hemos creado

$ git checkout ramaPrueba

Nota * git checkout -b ramaPrueba(nos crea una rama y nos lleva a la rama creada)

Y si hemos hecho cosas que nos importa de la ramaPrueba y nos interesa aplicarlos basta con hacer un git merge

$ git merge
